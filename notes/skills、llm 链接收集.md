# Obsidian skills

## 官方技能
```
npx skills add https://github.com/kepano/obsidian-skills 
``` 
## 基于 Karpathy 的 LLM Wiki 模式
```
npx skills add https://github.com/Ar9av/obsidian-wiki
```
## Obsidian 里生成 Canvas / Excalidraw / Mermaid
```
npx skills add https://github.com/axtonliu/axton-obsidian-visual-skills
```

## [LLM Wiki](https://github.com/nashsu/llm_wiki/blob/main/README_CN.md) 配套的 agent skill

- 触发约束：只有你明确说 `LLM Wiki/ 我的 wiki / 我的知识库` 

```
npx skills add https://github.com/nashsu/llm_wiki_skill.git --skill llm-wiki
```
